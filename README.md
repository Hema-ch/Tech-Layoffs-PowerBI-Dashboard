# 🌐 Global Tech Layoffs Analytics Dashboard (2020–2025)

## 📊 Project Overview
An interactive 5-page Power BI dashboard analyzing 581,705 layoff events 
across 1,745 companies in 50+ countries from 2020 to 2025.

## 🔍 Key Insights
- 2023 was the peak year — 170K+ layoffs in a single year
- 59.5% of layoffs came from Post-IPO companies
- Intel led with 37K layoffs, followed by Amazon (27K) and Meta (24K)
- USA accounted for 73% of all global tech layoffs
- Even companies with $1B+ funding cut 20-30% of their workforce

🎯 Strategic Recommendations
- HR teams should monitor overtime rates as early attrition signal
- Consumer & Retail industries need stronger workforce planning
- Workforce analytics should be a real-time boardroom tool

⚙️ Skills & Tools Used
- **Power BI Desktop** — 5-page interactive report
- **DAX** — TOPN, CALCULATE, FORMAT, DIVIDE, Dynamic titles
- **Power Query** — Data cleaning, null handling, custom columns
- **Data Visualization** — Map, scatter plot, treemap, donut, KPI cards
